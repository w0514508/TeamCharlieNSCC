# Cleaned Precipitation Data

This folder contains cleaned and processed monthly precipitation data prepared for the Capstone project.

- Data source: ECCC Historical Monthly Climate Data
  
- Station: Yarmouth (NS)
- Time coverage: 1940–2012
- Variable: Total Precipitation (mm)

- Station: Shearwater (NS)
- Time coverage: 1944–2007
- Variable: Total Precipitation (mm)

- Station: Sydney (NS)
- Time coverage: 1940–2012
- Variable: Total Precipitation (mm)

Data cleaning steps are documented in:
`notebooks/precipitation_monthly_cleaning_yarmouth.ipynb`
`notebooks/precipitation_monthly_cleaning_shearwater.ipynb`
`notebooks/precipitation_monthly_cleaning_sydney.ipynb`
